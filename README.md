# Python Data Analysis: NumPy & Pandas Foundations

This repository contains a practical exploration of core Data Science libraries in Python. The project focuses on efficient data manipulation using NumPy and structured data analysis using Pandas.

## 🛠️ Skills & Technologies
* **Python 3.x**
* **NumPy:** Multi-dimensional arrays, vectorized slicing, and data persistence (.npy).
* **Pandas:** Series manipulation, DataFrame engineering, and CSV handling.
* **Jupyter Notebook:** Interactive development and data visualization.

---

## 📋 Project Highlights

### 1. NumPy & NdArrays
* Demonstrated identical slicing behavior between Python lists and NumPy arrays.
* Managed actor/movie datasets using 2D NumPy arrays.
* Implemented **Boolean Indexing** to filter specific data subsets from arrays.
* Practiced data persistence by saving and reloading binary files from disk.

### 2. Pandas Series Operations
* Built custom `Series` objects with branded indices (e.g., Car Manufacturers).
* Used **Boolean Filtering** to extract specific string patterns from data.
* Cleaned and renamed metadata for improved dataset readability.

### 3. Advanced DataFrame Manipulation
* Engineered a multi-type DataFrame (String, Int, Boolean).
* Handled missing data by programmatically managing **NaN** values.
* Calculated complex row-level logic using the `.apply()` method with custom functions.
* Performed data sorting and identified extreme values (max/min) within the dataset.

---

## 📂 Project Structure
* `assignment 2- Data Processing.ipynb`: The primary analysis notebook.
* `*.npy`: Binary files generated during NumPy data export tasks.
* `*.csv`: Cleaned data exported from the Pandas DataFrame.
